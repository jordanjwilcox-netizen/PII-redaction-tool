# Privacy Redaction Tool

A local AI-powered privacy redaction desktop app built on OpenAI Privacy Filter

## Features
- Redacts emails, phone numbers, SSNs
- Supports .txt and .docx files
- Preserves Word formatting
- Detects PII using AI + regex fallback

## Planned Features
- DOB redaction
- Excel support
- PDF OCR redaction
- UI toggle controls

## Run

```bash
pip install -r requirements.txt
python app.py
